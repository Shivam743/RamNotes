# My Notes App

A simple note-taking app made with HTML, CSS, and JavaScript.

## What it does

- Click "Add Note" to create a new note
- Type anything in the note box
- Click "Delete" to remove a note
- Notes are yellow sticky notes style

## How to use

1. Download all 3 files
2. Put them in the same folder
3. Open `index.html` in your browser
4. Start adding notes!

## Files

- `index.html` - The main page
- `style.css` - Makes it look nice
- `script.js` - Makes the buttons work

## Note

Your notes will disappear when you close the page. This app does not save notes.

## Made with

- HTML
- CSS
- JavaScript

No extra libraries needed!
